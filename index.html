HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element contains information like copyright and contact info.
  • Look up a defintion for "semantic elements."

CSS

  • Whereas HTML strictly handles the structure and content of a webpage, CSS defines the colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • the padding, while transparent, is located inside the element, whereas the margin is the transparent buffer outside an element.
  • The display property assigned with the block value assigns each image to start on a new line.
  • The period preceding a class designates this as a class selector.
